Descrições das Tarefas

Tarefa 1: Posicionar o carro na rampa

Responsável: Marcelo

Pegar a chave do carro na sala, ligar o carro na garagem, colocar o carro na rampa, certificar que o freio de mão está acionado, que o carro está engatado e que os vidros estão fechados, desligar o carro, abrir as 4 portas, guardar a chave no bolso.

Voltar para Tabela

Tarefa 2: Buscar a mangueira

Responsável: Matheus

Ir até o depósito, pegar a mangueira com gatilho, levar a mangueira até a rampa, colocar a mangueira na torneira, ligar a torneira.

Voltar para Tabela

Tarefa 3: Buscar material de limpeza

Responsável: Ricardo

Ir até o depósito, pegar os seguinte items, levá-los até a rampa e deixar em local de fácil acesso e protegido:

  • Shampoo de lavar carro
  • Balde
  • 7 Panos:
    • 3x ensaboar o carro
    • 1x limpar os vidros
    • 1x passar silicone no painel e portas
    • 1x limpar porta-luvas
    • 1x verificar o óleo
Voltar para Tabela

Tarefa 4: Buscar material de limpeza

Responsável: Vitor

Ir até o depósito, pegar os seguinte items, levá-los até a rampa e deixar em local de fácil acesso e protegido:

  • Saquinho de lixo
  • Pretinho
  • Espuma para passar pretinho
  • Silicone
  • Limpa vidros
  • Aromatizante Automotivo
Voltar para Tabela

Tarefa 5: Limpar os tapetes

Responsável: Marcelo

Retirar os 4 tapetes do carro, as portas devem ser fechadas após retirar o respectivo tapete, exceto a porta do motorista que será fechada pela pessoa responsável por verificar o extintor, pedir para quem estiver com a mangueira jogar água, sacudir os tapetes para retirar o excesso de água, colocar os tapetes para secar no varal ao sol.

Voltar para Tabela

Tarefa 6: Jogar água nos tapetes e balde

Responsável: Matheus

Colocar água até três quartos do balde, jogar água nos tapetes do carro após retirados do mesmo por outra pessoa.

Voltar para Tabela

Tarefa 7: Diluir o shampoo nos baldes

Responsável: Ricardo

Esperar quem estiver com a mangueira jogar água no balde, jogar o shampoo automotivo e misturar até diluir.

Voltar para Tabela

Tarefa 8: Trocar o lixinho

Responsável: Vitor

Retirar o lixinho do câmbio pela porta do motorista, jogar na lata de lixo, colocar o saquinho de lixo novo.

Voltar para Tabela

Tarefa 9: Verificar extintor

Responsável: Vitor

Retirar o extintor debaixo do banco do motorista, verificar se está dentro do prazo de validade, caso não esteja deixar o extintor fora do carro para avisar o proprietário na retirada do carro, caso contrário colocar o extintor debaixo do banco e fechar a porta do motorista.

Voltar para Tabela

Tarefa 10: Jogar água no carro

Responsável: Matheus

Jogar água para molhar, esperar as outras pessoas ensaboarem o carro, enxaguar com água. Repetir esse processo para as seguintes partes de fora do carro, nessa ordem:

  1. Teto
  2. Vidro
  3. Capô
  4. Porta-malas
  5. Para-choques
  6. Rodas
Voltar para Tabela

Tarefa 11: Ensaboar

Responsável: Marcelo

Pegar um pano. Molhar o pano no balde com a solução de shampoo, esperar quem estiver com a mangueira jogar água para molhar, ensaboar o carro, esperar quem estiver com a mangueira enxaguar com água. Repetir esse processo para as seguintes partes de fora do carro, nessa ordem:

  1. Teto
  2. Vidro
  3. Capô
  4. Porta-malas
  5. Para-choques
  6. Rodas

No final colocar o pano no balde.

Voltar para Tabela

Tarefa 12: Ensaboar

Responsável: Vitor

Pegar um pano. Molhar o pano no balde com a solução de shampoo, esperar quem estiver com a mangueira jogar água para molhar, ensaboar o carro, esperar quem estiver com a mangueira enxaguar com água. Repetir esse processo para as seguintes partes de fora do carro, nessa ordem:

  1. Teto
  2. Vidro
  3. Capô
  4. Porta-malas
  5. Para-choques
  6. Rodas

No final colocar o pano no balde.

Voltar para Tabela

Tarefa 13: Ensaboar

Responsável: Ricardo

Pegar um pano. Molhar o pano no balde com a solução de shampoo, esperar quem estiver com a mangueira jogar água para molhar, ensaboar o carro, esperar quem estiver com a mangueira enxaguar com água. Repetir esse processo para as seguintes partes de fora do carro, nessa ordem:

  1. Teto
  2. Vidro
  3. Capô
  4. Porta-malas

No final colocar o pano no balde.

Obs: Esta pessoa não ajudará as outras duas a ensaboar a parte dos para-choques e rodas, pois nesse momento buscará o aspirador de pó.

Voltar para Tabela

Tarefa 14: Buscar o aspirador

Responsável: Ricardo

Ir até o depósito, pegar o aspirador e a extensão, levar o aspirador até a rampa, ligar o aspirador na extensão, ligar a extensão na tomada da garagem.

Voltar para Tabela

Tarefa 15: Passar pretinho nos pneus

Responsável: Marcelo

Pegar o pretinho e a espuma, passar nas laterais dos pneus usando a espuma cuidadosamente, repetir para os quatro pneus. Colocar o pretinho e a espuma devolta no lugar de fácil acesso e protegido.

Voltar para Tabela

Tarefa 16: Verificar níveis d'água

Responsável: Matheus

Abrir o capô, fixar o capô na posição aberta, verificar o nível de água do limpador de para-brisa, verificar o nível de água do radiador, caso seja necessário completar os reservatórios de água até o nível recomendado. Deixar o capô aberto.

Voltar para Tabela

Tarefa 17: Passar silicone no painel e no forro das portas

Responsável: Vitor

Pegar um pano e o silicone. Abrir a porta, colocar um pouco de silicone no pano, passar no forro da porta (parte de dentro), repitir para as quatro portas. Deixar as portas abertas. Colocar silicone no pano, passar no painel do carro. Colocar o silicone devolta no lugar de fácil acesso e protegido. Colocar o pano no balde.

Voltar para Tabela

Tarefa 18: Guardar a mangueira

Responsável: Matheus

Desligar a torneira, esvaziar a mangueira, enrolar a mangueira, levar a mangueira até o depósito, guardar a mangueira no depósito.

Voltar para Tabela

Tarefa 19: Aspirar o interior do carro

Responsável: Ricardo

As portas já estarão abertas. Ligar o aspirador de pó, passar o aspirador de pó no carpete, no teto e nos bancos do carro. Tomar cuidado para não atrapalhar quem estiver limpando os vidros.

Voltar para Tabela

Tarefa 20: Limpar os vidros por dentro

Responsável: Vitor

As portas já estarão abertas. Pegar um pano e o limpa vidros. Passar um pouco limpa vidros no pano, passar o pano nos vidros do lado de dentro do carro. Repetir para os vidros das quatro portas, para-brisa e vidro traseiro. Tomar cuidado para não atrapalhar quem estiver aspirando o pó. Colocar o limpa vidros devolta no lugar de fácil acesso e protegido. Colocar o pano no balde.

Voltar para Tabela

Tarefa 21: Colocar aromatizante

Responsável: Marcelo

Pegar o aromatizante automotivo, colocar o aromatizante na saída de ar do painel.

Voltar para Tabela

Tarefa 22: Colocar os tapetes no carro

Responsável: Marcelo

Retirar os tapetes do varal, certificar-se que estão secos, colocar os tapetes no carro, fechar as portas do carro. deixar a porta do passageiro por último pois outra pessoa estará limpando o porta-luvas ao mesmo tempo.

Voltar para Tabela

Tarefa 23: Guardar os produtos de limpeza

Responsável: Matheus

Retirar os panos do balde, dar os panos para o responsável por lavá-los, esvaziar o balde jogando a água com shampoo em um ralo. Jogar a espuma de passar pretinho fora se estiver muito suja. Verificar se todos produtos estão devidamente fechados e secos, guardar os seguintes produtos no depósito:

  • Shampoo automotivo
  • Balde
  • Pretinho
  • Espuma de passar pretinho
  • Silicone
  • Limpa vidros
Voltar para Tabela

Tarefa 24: Aspirar o porta-malas

Responsável: Ricardo

Abrir o porta-malas. Aspirar o interior do porta-malas. Fechar o porta-malas.

Voltar para Tabela

Tarefa 25: Limpar o porta-luvas

Responsável: Vitor

Abrir o porta-luvas, retirar o conteúdo e colocar no banco, limpar o porta-luvas, colocar o conteúdo devolta ao porta-luvas, fechar o porta-luvas.

Voltar para Tabela

Tarefa 26: Verificar óleo

Responsável: Marcelo

Verificar o óleo do motor, caso seja necessário trocar avisar o proprietário ao retirar o carro, pois é necessária autorização para realizar esse serviço. Desafixar o capô. Fechar o capô.

Voltar para Tabela

Tarefa 27: Limpar o saco do aspirador

Responsável: Ricardo

Desligar o aspirador, retirar o saco de pó, jogar o pó na lixeira, colocar o saco de pó devolta no aspirador.

Voltar para Tabela

Tarefa 28: Lavar os panos

Responsável: Vitor

Pegar os panos do balde com quem for o responsável por guardar o balde. Lavar os panos no tanque da lavanderia. Colocar os panos no varal para secar.

Voltar para Tabela

Tarefa 29: Guardar o aspirador

Responsável: Ricardo

Retirar a extensão da tomada, desconectar o aspirador da extensão, levar o aspirador e a extensão devolta ao depósito.

Voltar para Tabela

Tarefa 30: Guardar o carro na garagem

Responsável: Marcelo

Verificar se todos vidros e portas estão fechados. Pegar a chave no bolso. Ligar o carro. colocar o carro dentro da garagem. Guardar a chave na sala.

Voltar para Tabela
